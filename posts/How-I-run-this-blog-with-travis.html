How I run this blog with travis

  • Vue.js

I did a fork of this blog from ktquez/vuepress-theme-ktquez-starter . I tweaked it a little bit to build it with https://travis-ci.org for free.

Here is what I did:

I use two repositories: Blog and op07n.github.io.

The Blog repository is used to upload the new posts, and the other is automatically generated for travis-ci.

I created a new repository called op07n.github.io and added a few files in Blog repository:

.travis.yml

language: node_js
node_js:
    - 8.12.0
cache: 
    directories:
        - node_modules
script:
    - chmod u+x deploy.sh
    - yarn deploy-ci
branch: master

deploy.sh

#!/usr/bin/env sh
set -e

yarn build

cd src/.vuepress/dist

git init
git add -A # git add --all
git commit -m 'deploy'

git push -f https://${GH_TOKEN}@github.com/op07n/op07n.github.io.git master

cd -
